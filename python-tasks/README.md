## Задача 1: Расчет расходов на интернет

**Краткое описание задачи:**  
Пользователь имеет тарифный план с фиксированной платой и включенным объёмом интернет-трафика. Если пользователь превысит лимит, каждый дополнительный мегабайт оплачивается отдельно.  
Цель — рассчитать **общую стоимость интернета** для следующего месяца на основе этих данных.

**Данные для расчёта:**  
- A — стоимость тарифа в рублях  
- B — включенный объём трафика в мегабайтах  
- C — стоимость каждого лишнего мегабайта  
- D — планируемый объём интернет-трафика
