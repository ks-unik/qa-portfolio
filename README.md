## 📌 Описание папок

### Manual Testing
Каждое тестовое задание содержит:
- `test-cases/` – тест-кейсы (.md, .xlsx или .csv)  
- `bug-reports/` – баг-репорты с описанием, скриншотами и шагами воспроизведения  
- `README.md` – краткое описание проекта, условия и результаты

### Automation
Папка для будущих автоматизированных тестов:
- `src/` – исходники тестов  
- `configs/` – конфиги и настройки  
- `reports/` – отчёты, скриншоты, логирование

### Coding Tasks
- Задачи из учебных проектов и практики Python/Java/etc  
- Каждая задача в отдельной папке для удобства

### Docs
Шаблоны тест-кейсов, чек-листы, баг-репорты, шпаргалки и руководства

### UI Showcase
Примеры интерфейсов, HTML/CSS, скриншоты визуальных проектов
