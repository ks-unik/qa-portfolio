# Тестовое задание для amoCRM

В этой папке находится выполненное тестовое задание для вакансии QA в amoCRM.

## Описание
Были исследованы различные разделы сайта [amoCRM](https://www.amocrm.ru/).  
Каждый найденный баг оформлен с указанием:
- Краткого описания (Summary)
- Подробного описания (Description)
- Сообщения об ошибке (Error)
- Шагов для воспроизведения (Steps to Reproduce)
- Уровня серьёзности (Severity) и приоритета исправления (Priority)
